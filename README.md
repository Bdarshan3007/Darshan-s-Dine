# Darshan's Dine 🍽️

A modern, responsive web application for an online restaurant ordering system with AI-powered recommendations.

## Features

✨ **Core Features**
- 🏠 **Home Page** - Attractive landing page with restaurant information
- 📋 **Menu** - Browse and explore restaurant offerings
- 🛒 **Shopping Cart** - Add/remove items and manage orders
- 💳 **Checkout** - Secure payment and order confirmation
- 📧 **Contact** - Customer support and inquiries
- 🤖 **AI Recommendations** - Personalized dish suggestions based on preferences
- 🧾 **Invoice** - Digital receipt and order history

## Technology Stack

- **Frontend**: HTML5, CSS3
- **Styling**: Responsive CSS with mobile-first design
- **Features**: Dynamic cart management, AI recommendations

## Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- No server installation required for basic functionality

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Bdarshan3007/Darshan-s-Dine.git
   cd darshan-s-dine
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser, or
   - Use a local server (recommended):
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx http-server
     ```

3. **Access the application**
   - Navigate to `http://localhost:8000` (or your server port)
